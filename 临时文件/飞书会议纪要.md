- Skills 背景与定义
  - 背景介绍
    - 业务驱动：kostja 于 2022 年 8 月进入一家 AI 公司做营销和 SEO 工作，2024 年 10 月离职后从事增长和 SEO 咨询，后将积累的专业知识打包成 skills。
    - 发展历程：skills 最初是零散的 markdown 文件和人工编写的内容导向知识，随着生态发展逐渐完善。
  - 定义说明
    - 与文章区别：文章是给人读的，skills 是给 agent 读的，包含更多操作细则，能解决大模型缺少上下文时执行空泛的问题。
    - 规范开发：Markdown 格式的 skill 文档由 Anthropic 先开发，其他 agent 和大模型厂商跟进，其开发与 MCP 相关，很多 skills 由 AI 上下游产品官方发布。
    - 与普通 Markdown 区别：所有 skill 都是 Markdown，但并非所有 Markdown 都是 skill。skill 可自动调用，符合 Anthropic 规则的 skill 会被 Vocare Labs 网站收录。
- Skills 使用场景
- 个人开发者场景：个人开发者搭建的产品若营销或商业化不友好，可使用 skills 减少钻研营销内容的时间，如生成符合营销要求的 landing page 文案。
- AI 产品团队场景
  - 技术优化：国内建站团队前端或全栈人员 SEO 知识空缺，可使用 skills 进行 technical 优化。
  - 团队协作：AI CODING 流行，可通过 skills 将工程团队和运营团队分开，让运营团队独立开发网站内容发版。

- Skills 安装与使用
  - 安装方法：可通过命令行将 skills 安装成项目级或平台级，kostja 的 skills 每天更新，可使用对应指令获取最新版本。
- 使用平台
  - 通用支持：所有有对话交互框的 AI 工具都能使用 skills，但效率较低，建议使用 CO AI coding 平台，大部分平台都支持，如龙虾平台对 skills 支持良好。
  - 跨平台实施：skills 可跨平台使用，安装在不同平台无需修改即可使用。

- Marketing Skills 介绍
  - 内容构成
    - 核心内容：基于 SEO 场景，包含 144 个 skills，按一定分类规则排列，SEO、pages 和组件是核心内容，相关内容单独分类。
    - 具体技能：涵盖关键词调研、竞品调研、程序化 SEO、寄生 SEO 等，还涉及页面策略、组件设计、文案生成等实践内容。
  - 使用方式
    - 触发调用：在提示词中提及 skills 描述的关键词，即可自动调用对应技能，技能中还贴心添加了介绍。
    - 结合上下文：使用 skills 时需添加产品上下文，如品牌调性、受众人群、网址、关键词等，以达到最佳效果。

- 实际操作演示
  - 项目上下文
    - 文档生成：kostja 以 Polo 网站为例，展示了按模板生成的项目上下文文档，包含产品定位、商业模式、功能等通用信息，且文档会互相引用。
    - 协作方式：认为通过 agent 拉平甲乙双方信息是较好的协作方式，可让 AI 生成文档以避免遗漏。
  - 页面生成
    - 调用技能：使用 use case 页面生成技能，结合通用 skill 和部分上下文生成文档，虽不全面但比无技能时好，后期可添加更多项目上下文优化。
    - 页面效果：生成的页面从 SEO 角度看可用，但需添加其他技能或工具生成更炫酷的页面，还会生成与 SEO 相关的其他内容。
- 生态思考与问答环节
  - 生态思考
    - 多模态发展：当前 agent 的 skills 多用于 coding agent，多模态技能较少，但已有平台在添加。
    - 营销一致性：营销工作应基于产品上下文沉淀通用技能，在同一 agent 中完成，可减少部门协作摩擦，降低迁移成本。
  - 问答环节
    - 增长内容：kostja 表示做增长主要是出大方向策略，协调甲方团队实施落地并评估效果、提升效率。
    - SEO 相关：认为谷歌只看内容价值，做 SEO 需保证内容对用户、搜索引擎和大模型友好且可见；快速了解 SEO 可网上搜索或买教材，使用 skills 落地更快但可读性不如文章。
    - 工具建议：不推荐 WordPress 建站，认为其设计丑且麻烦；V0 难用，推荐 lovable 或国内性价比高的平台；Spotify 做电商平台和 Shopify CMS 不太好用；cursor 可用于写前端和内容。
    - 其他问题：对于淘宝 SEO 代运营，建议查看其之前代运营公司详情和流量数据；国内 SEO 可关注百度百家号、知乎、腾讯新闻等平台；可将运营动作沉淀为 skills 并迭代。
