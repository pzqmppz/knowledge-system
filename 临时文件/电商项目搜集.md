以下是我在 GitHub 上检索到的**比较新的电商相关开源项目**（重点筛选 2025-2026 年仍活跃维护、最近有更新记录的平台）。我优先选择了基于 “ecommerce” 话题的热门仓库（这些项目大多在 2026 年 3 月仍有 commit 更新，属于“比较新”的活跃项目），同时补充了国内特色的企业级系统。

这些项目大多采用现代技术栈（如 headless、API-first、多端支持），适合学习、二次开发或生产使用。我按 GitHub Stars 粗略排序，并附上关键信息：

### 国际主流活跃电商平台（推荐优先看这些，社区成熟）
1. **medusajs/medusa**  
   - Stars: 32.3k  
   - 语言：TypeScript  
   - 最近更新：2026 年 3 月 11 日  
   - 亮点：全球最灵活的 commerce 平台，支持 headless、无头架构，适合快速构建自定义电商。Node.js 生态，插件丰富。  
   - 链接：https://github.com/medusajs/medusa

2. **bagisto/bagisto**  
   - Stars: 26.1k  
   - 语言：PHP (Laravel)  
   - 最近更新：2026 年 3 月 11 日  
   - 亮点：免费开源 Laravel 电商平台，支持多商户（multi-vendor）、headless 模式，适合中大型店铺。  
   - 链接：https://github.com/bagisto/bagisto

3. **saleor/saleor**  
   - Stars: 22.7k  
   - 语言：Python  
   - 最近更新：2026 年 3 月 11 日  
   - 亮点：高性能 headless commerce API，GraphQL 支持，适合需要可组合（composable）架构的企业。  
   - 链接：https://github.com/saleor/saleor

4. **spree/spree**  
   - Stars: 15.3k  
   - 语言：Ruby  
   - 最近更新：2026 年 3 月 11 日  
   - 亮点：API-first 企业级平台，原生支持 B2B、多商户、多租户。  
   - 链接：https://github.com/spree/spree

5. **vercel/commerce**  
   - Stars: 13.9k  
   - 语言：TypeScript (Next.js)  
   - 最近更新：2026 年 2 月  
   - 亮点：Vercel 出品的现代前端电商模板，React/Next.js 驱动，适合快速原型或全栈电商前端。  
   - 链接：https://github.com/vercel/commerce

其他值得一提的活跃项目（同话题下）：**woocommerce/woocommerce**（WordPress 插件，10.2k stars）、**nopSolutions/nopCommerce**（.NET/C#，10k stars）、**magento/magento2**（企业级 PHP，12.1k stars），这些也都保持 2026 年活跃更新。

### 国内/中文特色推荐（特别适合中国开发者，多端支持强）
**tigshop/Tigshop**（强烈推荐）  
- Stars: 7.1k  
- 语言：Java（Spring Boot 3）/ PHP（ThinkPHP 8）双版本 + Vue 3 / Uni-app 前端  
- 最近更新：2025 年 11 月（仍活跃维护）  
- 亮点：**100% 开源企业级商城系统**，支持 H5、微信小程序、公众号、iOS、安卓、鸿蒙多端；内置秒杀、优惠券、积分、页面装修、分销、IM 客服；有单商户免费版 + 多商户 B2B2C、O2O 多门店、跨境多语言等独立版本。高并发、极易二次开发，无加密。技术栈非常新（Spring Boot 3 + Vue 3 + TS）。  
- Demo：PC https://demo.tigshop.cn / 管理后台同域名（账号 demo/demo123）  
- 链接：https://github.com/tigshop/Tigshop  
- 官网/文档：https://www.tigshop.com/（有完整安装教程和社区）
