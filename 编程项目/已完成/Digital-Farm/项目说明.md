# Digital-Farm 项目说明

> AI 驱动的自动化知识管理中间件系统

---

## 项目定位

**Digital-Farm** 是一个将外部信息（电子书、网页、文档）通过 AI 处理转化为结构化知识的自动化生产线。

### 核心愿景

- **私有化与掌控**：数据完全自主掌控
- **高度自动化**：仅需输入 URL 或上传文件
- **AI 增强提取**：使用 GLM-4.7-Flash / DeepSeek 双模型
- **知识分级管理**：自动区分"时效信息"与"基石知识"

---

## 工作流程

```
播种（用户填写 URL）
    ↓
发芽（状态标记为"🌱 播种中"）
    ↓
生长（后端自动提取 + AI 分析）
    ↓
收获（状态更新为"🍎 已入库"，发送通知）
```

## 技术架构

### 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | Next.js 14.2 + React 18 |
| 后端 | Next.js API Routes |
| AI 模型 | GLM-4.7-Flash / DeepSeek-V3 |
| 知识存储 | Notion API |
| 缓存/锁 | Upstash Redis |
| 日志 | Pino |
| 监控 | Prometheus |

### 部署方案

| 层级 | 平台 | 原因 |
|------|------|------|
| 前端展示 | Vercel | 快速、稳定 |
| 后端处理 | Docker | 无超时限制、完全可控 |

## 核心功能

### 1. 多格式文档解析

| 格式 | 支持程度 |
|------|----------|
| EPUB | ⭐⭐⭐⭐⭐ 完全支持 |
| MOBI/AZW3 | ⭐⭐⭐⭐ 标准格式支持 |
| PDF | ⭐⭐⭐ 字体大小检测 |
| 网页 | ⭐⭐⭐⭐⭐ 智能去广告 |

### 2. AI 逻辑研磨器

- **去水存骨**：提取逻辑骨架
- **餐巾纸时刻**：抽象为核心图形
- **跨界迁移**：自动思考应用场景
- **稳定性评估**：区分时效与基石知识

### 3. 自动分级与评分

- **相关性评分**：1-10 分
- **行动优先级**：立即执行 vs 参考备用

---

## 处理模式

根据内容大小自动选择处理策略：

| 模式 | 条件 | 流程 |
|------|------|------|
| 完整处理 | < 50k tokens | 一次性处理 |
| 三层架构 | 50k-150k tokens | 快速扫描 → 关键章节 → 深度合成 |
| 快速扫描 | > 150k tokens | 仅分析目录 + 前 5 章 |
