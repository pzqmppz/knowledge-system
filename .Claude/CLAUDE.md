# 知识体系配置

> 这是本知识体系的 Claude Code 配置文件

## 目录结构

```
知识体系/
├── 拆解报告/          # 书籍、论文、文章、口播稿的深度拆解
│   ├── 书籍/
│   ├── 论文/
│   ├── 文章/
│   └── 口播稿/        # 播客/视频口播稿解读 (voiceover-xray)
├── 书籍管理/          # 素材的入库和管理
│   ├── Box/           # 临时入库区
│   │   ├── 书库/
│   │   ├── 论文/
│   │   ├── 文章/
│   │   └── 播客口播稿/
│   ├── 阅读中/        # 正在阅读的内容
│   └── 已归档/        # 已完成的内容
├── .Claude/           # Claude Code 配置
│   ├── skills/        # 本知识体系专用的 skills
│   ├── schemas/       # 数据结构定义
│   └── scripts/       # 辅助脚本
└── INDEX.md           # 主索引
```

## 可用 Skills

| Skill | 命令 | 用途 |
|-------|------|------|
| book-xray | `/book-xray` | 书籍深度拆解 |
| paper-xray | `/paper-xray` | 论文解读 |
| article-xray | `/article-xray` | 文章解读 |
| voiceover-xray | `/voiceover-xray` | 口播稿深度解读 |

## 工作流

### 新增素材

1. 下载素材放到 `书籍管理/Box/` 对应目录
2. 更新 `书籍管理/Box/INDEX.md`
3. 使用对应的 skill 进行拆解

### 拆解流程

- **书籍**: `/book-xray <文件路径>` → 生成到 `拆解报告/书籍/`
- **论文**: `/paper-xray <文件路径>` → 生成到 `拆解报告/论文/`
- **文章**: `/article-xray <URL>` → 生成到 `拆解报告/文章/`
- **口播稿**: `/voiceover-xray <文本/文件路径>` → 生成到 `拆解报告/口播稿/`

## 索引维护

拆解完成后，系统会自动更新：
- `拆解报告/INDEX.md` - 主索引
- `拆解报告/<类型>/INDEX.md` - 分类索引
- `书籍管理/Box/INDEX.md` - 入库索引
