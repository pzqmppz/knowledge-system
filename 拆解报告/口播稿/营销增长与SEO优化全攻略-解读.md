### 核心知识/价值点

#### 商业/业务类核心价值

**主流程图谱**：
```
营销自动化流程：
产品上下文 + Skills → Agent 执行 → 生成营销物料 → 上线 → 监测效果
     ↓              ↓            ↓              ↓        ↓
  PRD/定位      选择对应       Landing/       发布    GSC分析
               技能包         文章/页面
```

**核心技巧提炼**：
1. **上下文驱动**：Skills 是通用知识，必须结合产品上下文才能生效
2. **迭代优化**：生成效果不满意 → 反馈给 AI → 修改 Skills → 再生成
3. **人机协作**：策略 AI 出，人工审核；执行 AI 做，人工迭代
4. **跨平台复用**：Skills 写一次，Cursor/Replit/Web Coding 都能用

**隐含假设**：
- 你的产品/服务有价值，只是缺少营销能力
- 你愿意学习新工具（Cursor 等 AI IDE）
- 你接受 AI 生成内容需要人工审核的事实

#### 技术闪光点

**Skills vs 文章的本质区别**：
- 文章 = 给人读的叙述性内容
- Skills = 给 Agent 读的执行性指令

**价值主张**：
- 通用 AI 生成内容 = 60 分
- 专业人员做 = 80 分
- Skills + 产品上下文 = 80 分 + 可复用 + 可扩展
