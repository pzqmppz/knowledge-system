## 餐巾纸 | Napkin

### 公式

```
+----------------------------------------------------------+
|                                                          |
|   观察 → 感受 → 需要 → 请求                              |
|   Observation · Feeling · Need · Request               |
|                                                          |
|   他人言行 → 不是我感受的原因                            |
|   我的感受 → 根源于我的需要                             |
|   批评指责 → 隐含未满足的需要                            |
|                                                          |
+----------------------------------------------------------+
```

非暴力沟通的四个要素构成了一个完整的循环：表达自己的观察、感受、需要和请求；同时倾听他人的观察、感受、需要和请求。

### 一句话

通过专注于彼此的观察、感受、需要和请求，而非评判、指责和命令，我们能够建立情意相通的连接，实现乐于互助的关系。

## 第一轮：骨架扫描 | Skeleton Scan

"这本书在说什么"

### 核心问题

为什么我们天生热爱生命、乐于互助，却常常用伤害性的方式沟通？如何让爱融入生活，建立情意相通的人际关系？

### 核心答案

通过学习非暴力沟通的四个要素（观察、感受、需要、请求），我们可以转变谈话和聆听的方式，不再条件反射式地反应，而是专注于彼此的观察、感受和需要，从而实现情意相通、乐于互助的关系。

### 章节骨架

1. 让爱融入生活 - 非暴力沟通的四个要素
2. 是什么蒙蔽了爱？ - 异化的沟通方式
3. 区分观察和评论 - 第一个要素
4. 体会和表达感受 - 第二个要素
5. 感受的根源 - 第三个要素（需要）
6. 请求帮助 - 第四个要素
7. 用全身心倾听 - 倾听他人
8. 倾听的力量 - 倾听的力量
9. 爱惜自己 - 非暴力沟通的应用
10. 充分表达愤怒 - 愤怒的处理
11. 运用强制力避免伤害 - 强制力的使用
12. 重获生活的热情 - 解除抑郁
13. 表达感激 - 感激的表达

### 论证结构

**案例型 + 方法论型**：通过大量真实案例（家庭、学校、职场、国际冲突）来阐述非暴力沟通的原理和应用，然后给出具体的操作方法。

## 第二轮：血肉解剖 | Deep Dissection

"凭什么这么说"

### 论证链

```
人性的基本假设（人天生热爱生命、乐于互助）
        ↓
异化沟通方式蒙蔽了爱（道德评判、比较、回避责任、强人所难）
        ↓
非暴力沟通的四个要素（观察、感受、需要、请求）
        ↓
应用于表达自己 + 倾听他人
        ↓
建立情意相通、乐于互助的关系
```

### 关键证据

1. **集中营案例**：作者在巴勒斯坦难民营被骂"谋杀犯"，通过倾听对方的感受和需要，最终被邀请去家中吃晚餐

2. **学校调解案例**：教师与校长长期无法沟通，通过区分观察和评论（"你是个大嘴巴" vs "你在会议中讲了20分钟童年故事"），最终改善关系

3. **临终关怀案例**：护士通过非暴力倾听，帮助临终病人和妻子建立最后的深度连接

### 隐形假设

- 人性本善，而非性恶论
- 感受的根源在于自身需要，而非他人行为
- 所有的批评、指责、攻击都是表达需要的方式
- 情意相通后，人们自然会乐于互助

### 边界条件

- 非暴力沟通不是权宜之计，而是持续的生活方式
- 不适用于紧急危险情况（需要立即保护自己）
- 在对方极度情绪化时，可能需要先调整环境再沟通
- 在某些文化环境中，直接谈论感受可能被视为不恰当

"还能怎么用"

### 作者盲点

1. **文化差异**：书中案例多来自西方语境，在东方文化（含蓄、面子文化）中应用可能需要调整
2. **权力不对等**：在严重权力不对等的关系中（如职场上下级、婆媳关系），单方面使用非暴力沟通可能无济于事
3. **系统性暴力**：非暴力沟通关注个人层面的转变，但忽略了结构性暴力的问题

### 行动触发

读完这本书后，可以做的不同的事：

1. **每天练习一次"我感到...因为我需要..."** 的句式
2. **在冲突时刻停顿 3 秒**，区分观察和评论
3. **倾听他人时**，不急于提建议，先确认是否理解了对方的感受和需要
4. **对自己说"不"时**，关注自己真正需要的是什么
5. **表达感激时**，说出对方做了什么、自己感受如何、什么需要得到了满足

## 核心概念归档

### 概念 1：异化的沟通方式

- **定义**：蒙蔽爱、导致疏远和伤害的沟通方式
- **四种类型**：
  1. 道德评判：用价值观评判人（"你太自私了"）
  2. 比较：与他人比较蒙蔽爱意
  3. 回避责任：用"不得不"、"你让我"淡化个人责任
  4. 强人所难：威胁式要求（"如果不...就..."）

### 概念 2：非暴力沟通四要素

1. **观察**：清楚表达观察结果，不判断或评估
   - 好的观察："欧文在5场比赛中没有进一个球"
   - 差的观察："欧文是个差劲的前锋"

2. **感受**：表达情绪状态（受伤、害怕、喜悦、开心等）
   - 区分感受和想法："我感到受伤" vs "我觉得你不重视我"

3. **需要**：感受的根源在于自身需要
   - 句式："我感到...因为我需要..."
   - 基本需要：自由选择、庆祝、言行一致、滋养身体、玩耍、情意相通、相互依存

4. **请求**：具体的、可执行的行动请求
   - 区分请求和命令：对方可以说"不"的才是请求

### 概念 3：听到不中听的话时的四种选择

1. 认为自己犯了错 → 内疚、惭愧
2. 指责对方 → 恼怒
3. 了解自己的感受和需要 → 诚实表达
4. 体会他人的感受和需要 → 同理心

### 概念 4：全身心倾听

- **放下**已有的想法和判断
- **用心体会**他人的观察、感受、需要、请求
- **避免**：建议、比较、说教、安慰、回忆、否定、同情、询问、辩解、纠正
- **佛教格言**："不要急着做什么，站在那里"

### 概念 5：个人成长的三个阶段

1. **"情感的奴隶"**：认为自己有义务让他人快乐
2. **"面目可憎"**：拒绝考虑他人的感受和需要
3. **"生活的主人"**：出于爱而非恐惧/内疚帮助他人
