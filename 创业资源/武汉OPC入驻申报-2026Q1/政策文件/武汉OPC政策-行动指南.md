# 武汉OPC政策行动指南

> 政策原文：[武汉市支持人工智能OPC创新发展若干措施](https://www.wuhan.gov.cn/zwgk/xxgk/zfwj/gfxwj/202602/t20260214_2730572.shtml)
> 生效日期：2026年3月18日 | 有效期至：2028年12月31日

---

## 一、OPC定义

> 由具备AI应用开发与整合能力的人才或微型团队创办，以大模型为基础，AI编程、智能体等为核心工具，在研发、生产、运营等全流程实现效率超线性放大的"AI＋超级个体"新型创业实体。

---

## 二、我的项目定位

**AI驱动的内容生产与运营一体化平台**

| 层级 | 内容 | 政策对应 |
|------|------|----------|
| 第一层 | 视频/音频/文字的AI工作流化 | 模型工具研发、算力需求 |
| 第二层 | AI辅助运营与自动化营销 | 垂直领域应用、场景对接 |

## 三、可申请政策汇总

### 核心相关（优先申请）

| 政策 | 补助金额 | 关键条件 |
|------|---------|---------|
| 算力服务券 | 最高20万/年，最长3年 | 算力费用50%补助 |
| 模型工具研发 | 最高100万 | 纳入市级重点研发计划 |

| 应用场景项目 | 最高100万 | 项目总投入30%补助 |

### 降低成本类

| 政策 | 内容 | 备注 |
|------|------|------|
| 免费算力 | 每年2000卡时 | 入驻OPC生态社区 |
| 租房优惠 | 2年每月2000元免租 | 按博士标准，需OPC身份 |
| 购房补贴 | 最高10万 | 1000元/㎡，限新建商品房 |
| 免费住宿 | 最长15天 | 武汉青年人才之家 |
| 公共服务包 | 最长2年免费 | 财务代账、税务、法律、专利 |

### 融资支持

| 政策 | 金额 | 条件 |
|------|------|------|
| 个人创业担保贷款 | 最高30万 | 10万以下免反担保 |
| 企业创业担保贷款 | 最高500万 | 最长2年 |
| 算力贷 | 最高1000万 | 80%风险补偿 |

## 四、待办清单

### 立即可做（现在-2026年3月）

- [ ] 关注"武创通"科创服务平台（OPC线上开放社区载体）https://www.whwct.com/home 联系电话 027-82852280
- [ ] 关注市科创局官网，获取后续实施细则
- [ ] 关注市经信局官网，了解算力平台建设进展
- [ ] 完成公司注册（大部分政策需要企业主体）
- [ ] 完善产品原型（至少一个可展示的工作流demo）
- [ ] 整理案例材料（将知识体系项目包装为应用案例）
- [ ] 关注东湖高新区/所在区的科创管理部门动态

### 政策生效后（2026年3月起）

- [ ] 申请入驻OPC生态社区
- [ ] 申报算力服务券

- [ ] 参加OPC创新创业大赛（每年发掘50+标杆项目）
- [ ] 通过"场景机会清单"对接应用场景
- [ ] 申报创业资助（最高100万）

## 五、关键问题待明确

1. **OPC身份认定流程** - 政策定义了OPC但未明确认定标准，需关注后续细则
2. **市级算力公共服务平台** - 尚未上线，需关注发布渠道
3. **OPC生态社区名单** - 2026年底建成3个，需关注各区招商信息
4. **"武创通"平台** - OPC线上社区载体，需关注上线时间

## 六、申请策略建议

### 定位包装

- 工具产品："AI内容工作流平台"
- 服务方案："企业AI营销解决方案"
- 两者结合更易获得"场景对接"支持

### 材料准备

1. 商业计划书 - 产品定位、目标市场、盈利模式
2. 技术说明 - 工作流架构、AI技术栈、创新点
3. 应用案例 - 知识体系项目作为内部孵化案例
4. 团队能力 - AI能力、内容能力、项目经验

### 核心思路

```
知识体系项目 → 内部孵化案例 → 申请材料 → 政策支持
```
