# 摘抄笔记

```

│   └── ...

```

## 使用方法

### 方式一：QuickAdd 插件（推荐）

1. 选中要摘抄的文字
2. 按 `Ctrl + Shift + Q`（需先配置）
3. 自动追加到当天摘抄文件

### 方式二：手动创建

按日期创建文件，格式：`YYYY-MM-DD-摘抄.md`
